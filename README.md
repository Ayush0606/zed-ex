Norrav Landing Page is a **Next.js-based frontend project** designed to demonstrate modern UI/UX practices, interactive animations, and responsive design.  
It serves as both a **portfolio showcase** and a **creative studio landing page**, highlighting services like:  

- Branding & Visual Identity  
- Digital Design & UI/UX  
- Motion Graphics & Animations  
- Creative Solutions for clients  

The project emphasizes:  
- **Clean & Minimal Design**: Focus on typography, whitespace, and layout.  
- **Interactive Animations**: Hover effects, scroll-based animations, and micro-interactions.  
- **Responsive & Mobile-First**: Optimized for all devices.

## ðŸš€ Installation

Kickstart your project with these steps:

1.**Clone the repository**
 ```bash
  git clone https://github.com/peyush-nuwal/norrav-landing-page.git
   ```

2.**Navigate to the project directory**

   ```bash
   cd norrav-landing-page
   ```

3.**Install dependencies**

   ```bash
   npm install
   ```
   > Use `yarn` or `pnpm` if preferred.


5.**Launch the development server**

   ```bash
   npm run dev
   ```

6.**Server is running on port 3000**

```bash
http://localhost:3000
```
